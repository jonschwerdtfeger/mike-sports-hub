# SportsHub Prototype Reference

## Product Goal

SportsHub is a personal sports command center that lets a fan feel completely caught up on all selected teams in one place.

## Approved Dashboard Sections

1. Today's Focus
   - Definition: What deserves attention right now
   - Shows live games first
   - If no live games, shows upcoming games
   - If no upcoming games, shows recent results

2. Team Spotlight Cards
   - Definition: Entry points into each team
   - One card per selected team
   - Shows record/status/last/next
   - Opens a team detail drawer

3. Upcoming Watchlist
   - Definition: Calendar-style upcoming games
   - Chronological schedule across all selected teams

4. Team Signals
   - Definition: One meaningful signal per team
   - One meaningful team-specific signal per team

5. Top Headlines
   - Definition: Best recent stories across selected teams
   - Relevant headlines across selected teams

6. Roster Wire
   - Definition: Personnel-related news only
   - Roster, injury, transaction, and personnel-related updates

## Approved Team Drawer Sections

1. Record / Standing / Status / Next Game
2. Standings
3. Last 5 / Next 5
4. Roster Pulse
5. Latest Signals

## Known Prototype Limitations

- Hard-coded Michael profile
- Direct ESPN dependency
- No authentication
- No database-backed team selection
- No shared provider cache
- Transaction detection is keyword-based
- News relevance still needs improvement
